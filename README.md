# ProductosTiendaOnline

## 📌 Descripción del proceso
Automatización del análisis diario de productos disponibles en una tienda online.  
El desarrollo fue realizado con la plantilla universal de **PIX RPA**.

---

## ▶️ Pasos para ejecución
1. Descargar el proyecto **ProductosTiendaOnline** en el equipo para la ejecución.  
2. Ejecutar el script de **Base de Datos** para poder guardar los productos obtenidos.  
   - Motor de BD utilizado: **SQL Server**.  
3. Tener instalada la extensión **PIX Studio Plugin**.  

---

## ⚙️ Requisitos
1. La automatización del formulario web se realiza en el navegador **Google Chrome**, se debe tener instalado.  
2. Tener creada la **Base de Datos** con la tabla `Productos`.
